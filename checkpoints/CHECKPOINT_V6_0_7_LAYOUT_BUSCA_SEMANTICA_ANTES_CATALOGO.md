# AVDC V6.0.7 — Layout da busca semântica antes do catálogo

## Objetivo

Ajustar a ordem visual dos quadros da tela principal.

## Decisão validada

- O teste de conexão da IA permanece validado e preservado.
- O quadro Motor de IA permanece separado da busca semântica.
- O quadro Busca semântica deve aparecer antes do quadro Catálogo inicial do índice.
- Quando a busca semântica for marcada, os controles abaixo relacionados à busca simples/catálogo continuam sendo desabilitados.

## Ordem visual esperada

1. Repositórios do AVDC
2. Motor de IA
3. Busca semântica
4. Catálogo inicial do índice
5. Busca simples

## Observação

Esta versão não altera a regra do teste de conexão: o botão testa os dados de IA já salvos no banco para o usuário logado.
